命令速查

按分类整理常用命令，支持搜索和一键复制。

npm

npm 包发布、本地静态服务等。

npm publish
发布 npm 包到 npm registry。
npx http-server -p 8080
在当前目录启动静态文件服务器，端口为 8080。

远程脚本

通过 curl 下载并执行远程脚本。

curl -fsSL https://example.com/sh/sh.sh | sudo bash
下载远程脚本并以 sudo 权限执行。 执行前建议审查代码。

Backblaze B2

对象存储同步。

b2 sync "C:\example\" "b2://example"
把本地 example 文件夹同步到 Backblaze B2 存储桶。

PAR2 校验 / 恢复

生成PAR2文件，用于文件损坏后的修复。

"C:\Program Files (x86)\MultiPar\par2j64.exe" c -rr5 -sn2048 -rf1 "${PWD##*/}.par2" *
使用 MultiPar 为当前目录文件生成 PAR2 校验和恢复文件。

文件校验

计算文件哈希值。

md5sum <文件名>
计算文件的 MD5 值，用于校验文件完整性。

Python 打包

PyInstaller 打包相关。

pyinstaller HexoDash.spec
根据 HexoDash.spec 配置文件打包 Python 项目。

Rust / Cargo / Tauri

Rust 清理、构建，以及 Tauri 开发和打包。

cargo clean
清理 Rust 项目的构建缓存和产物。
npm run tauri dev
以开发模式启动 Tauri 应用。
npm run tauri build
构建 Tauri 应用正式版本。
npm run tauri build -- --debug
构建 Tauri 应用 debug 版本。
RUSTFLAGS="-C target-feature=+crt-static" cargo build --release
使用静态 CRT 构建 Rust release 版本。
RUSTFLAGS="-C target-feature=+crt-static,+avx2" cargo build --release
构建 release 版本，启用静态 CRT 和 AVX2 指令集。
RUSTFLAGS="-C target-cpu=native -C target-feature=+avx2" npm run tauri build -- -- --features "avx"
针对当前 CPU 优化，启用 AVX2，并启用 Rust feature：avx。
npm run tauri icon -i ./logo.png
使用 logo.png 生成 Tauri 应用图标。

C++ 编译

使用 g++ 编译源码。

g++ sitemap.cpp -o sitemap.exe
编译 sitemap.cpp，并输出 sitemap.exe。

Cloudflare Wrangler

Workers 和 Pages 部署、日志查看。

wrangler tail
实时查看 Cloudflare Workers 日志。
wrangler deploy
部署 Cloudflare Worker。
wrangler pages deploy
部署 Cloudflare Pages 项目。

Windows 文件 / 目录

目录树、目录列表导出。

tree /F /A
以 ASCII 树状结构显示当前目录和文件。
dir E:\ /ad /b > 1.txt
列出 E 盘根目录下的文件夹名称，并输出到 1.txt。

Linux / Unix 系统

进程查看、权限设置、后台运行。

ps -aux
查看系统进程列表。
chmod +x name.sh
给 name.sh 添加可执行权限。
nohup ./example &
使用 nohup 在后台运行程序，关闭终端后仍继续运行。

正则表达式

文本处理和编辑器搜索替换。

^\s*$\n
匹配空行，常用于批量删除空白行。
